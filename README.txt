EB de Manhente — Página do Tempo (pack)

Ficheiros:
- tempo.html  (página principal)
- config.js   (config opcional Weather Underground)
- README.txt

Como usar:
1) Copia estes ficheiros para a pasta do teu site (GitHub Pages) onde está o resto.
2) Garante que o link para a página do tempo aponta para tempo.html.
3) Se quiseres puxar vento/rajada do Weather Underground:
   - abre config.js
   - preenche stationId (o ID da tua estação no WU)
   - nota: a apiKey fica pública em GitHub Pages (recomendado usar proxy).

Sem stationId, a página funciona na mesma (tempo + qualidade do ar via Open-Meteo).
